Data Entry & Basics (17 Techniques)

  1. What it does: Move quickly around your worksheet without using the mouse.

    1. Open your Excel worksheet.
    2. Use arrow keys to move cell by cell.
    3. Hold Ctrl + Arrow key to jump to the end of data ranges.

  2. What it does: Automatically fill sequences or repeat values.

    1. Enter the starting value in a cell (e.g., 1 or Jan).
    2. Hover over the bottom-right corner until the cursor becomes a small plus (+).
    3. Click and drag down or across to fill the series.

  3. What it does: Restrict data entry to valid inputs only.

    1. Select the cells where you want to apply validation.
    2. Go to Data → Data Validation → Data Validation.
    3. Choose the type of validation (e.g., whole number, list).
    4. Set the criteria and click OK.

  4. What it does: Let users select options from a predefined list.

    1. Select a cell or range.
    2. Go to Data → Data Validation → List.
    3. Enter the values or select a range containing the list.
    4. Click OK and test the dropdown.

  5. What it does: Automatically fills patterns based on example entries.

    1. Type the example pattern next to your data.
    2. Go to Data → Flash Fill, or press Ctrl + E.
    3. Excel fills remaining cells following the pattern.

  6. What it does: Keep row or column headers visible while scrolling.

    1. Select the cell below the row(s) and right of the column(s) to freeze.
    2. Go to View → Freeze Panes → Freeze Panes.

  7. What it does: Combine or separate cells for better layout.

    1. Select cells to merge and click Home → Merge & Center.
    2. To split, select merged cell → Home → Merge & Center → Unmerge Cells.

  8. What it does: Split data from one column into multiple columns.

    1. Select the column with data to split.
    2. Go to Data → Text to Columns.
    3. Choose Delimited or Fixed Width and follow the wizard.

  9. What it does: Highlight data that meets certain conditions.

    1. Select the range to format.
    2. Go to Home → Conditional Formatting → Highlight Cells Rules.
    3. Set the condition and choose formatting.

  10. What it does: Display numbers, dates, or text in a custom way.

    1. Select cells to format.
    2. Right-click → Format Cells → Number → Custom.
    3. Enter the desired format code (e.g., #,##0; "USD" #,##0).

  11. What it does: Organize data in a structured, filterable format.

    1. Select the range of data.
    2. Go to Insert → Table and click OK.
    3. Use table headers for sorting and filtering.

  12. What it does: Arrange and view only data that meets criteria.

    1. Select your data range.
    2. Go to Data → Sort or Filter.
    3. Apply criteria to organize your data.

  13. What it does: Assign a name to a cell or range for easier reference in formulas.

    1. Select the cell/range.
    2. Click in the Name Box, type a name, press Enter.
    3. Use the name in formulas instead of cell references.

  14. What it does: Apply formatting quickly using the keyboard.

    1. Ctrl + B for bold, Ctrl + I for italics, Ctrl + U for underline.
    2. Ctrl + Shift + $ for currency, Ctrl + Shift + % for percentage.

  15. What it does: Prevent accidental editing of important data.

    1. Go to Review → Protect Sheet.
    2. Set a password and select which actions are allowed.
    3. Click OK to protect your sheet.

  16. What it does: Quickly select specific types of cells (blanks, formulas, etc.).

    1. Press F5 → Special.
    2. Select the type of cell to highlight.
    3. Click OK to select them.

  17. What it does: Identify and correct errors in data.

    1. Look for small green triangles in cells.
    2. Click the warning icon → Select the appropriate correction.

Formulas & Calculations (18 Techniques)

  1. What it does: Calculate totals, averages, and extreme values.

    1. Select the cell for the result.
    2. Type =SUM(range) or =AVERAGE(range) or =MIN(range)/=MAX(range).
    3. Press Enter to see the result.

  2. What it does: Count numbers or all non-empty cells in a range.

    1. Type =COUNT(range) to count numeric cells.
    2. Type =COUNTA(range) to count all non-empty cells.
    3. Press Enter.

  3. What it does: Make decisions based on conditions.

    1. Select a cell to enter the formula.
    2. Type =IF(condition, value_if_true, value_if_false).
    3. Press Enter to apply.

  4. What it does: Handle multiple conditions in one formula.

    1. Type =IF(condition1, value1, IF(condition2, value2, value3)).
    2. Press Enter to see the result.

  5. What it does: Lookup values vertically in a table.

    1. Type =VLOOKUP(lookup_value, table_range, column_index, FALSE).
    2. Press Enter to find matching data.

  6. What it does: Lookup values horizontally in a table.

    1. Type =HLOOKUP(lookup_value, table_range, row_index, FALSE).
    2. Press Enter to find matching data.

  7. What it does: Advanced lookup combining INDEX and MATCH for flexibility.

    1. Type =INDEX(return_range, MATCH(lookup_value, lookup_range, 0)).
    2. Press Enter to retrieve the result.

  8. What it does: Extract specific text from cells.

    1. Use =LEFT(text, num_chars), =RIGHT(text, num_chars), or =MID(text, start_num, num_chars).
    2. Press Enter to extract desired text.

  9. What it does: Combine text from multiple cells.

    1. Type =CONCAT(A1,B1) or =TEXTJOIN(" ", TRUE, A1:A3).
    2. Press Enter to see combined text.

  10. What it does: Round numbers to desired decimal places.

    1. Type =ROUND(number, num_digits), =ROUNDUP(number, num_digits), or =ROUNDDOWN(number, num_digits).
    2. Press Enter to round.

  11. What it does: Get the absolute value of a number.

    1. Type =ABS(number).
    2. Press Enter to see the positive value.

  12. What it does: Insert current date and time or only date.

    1. Type =NOW() for date and time.
    2. Type =TODAY() for only the date.
    3. Press Enter to apply.

  13. What it does: Calculate loan payments based on interest, periods, and principal.

    1. Type =PMT(rate, nper, pv).
    2. Press Enter to calculate the payment.

  14. What it does: Combine multiple conditions logically.

    1. Type =AND(condition1, condition2) or =OR(condition1, condition2).
    2. Press Enter to get TRUE or FALSE.

  15. What it does: Replace errors with custom values.

    1. Type =IFERROR(formula, "Custom message").
    2. Press Enter to see either result or message.

  16. What it does: Reference a range offset from a starting point.

    1. Type =OFFSET(reference, rows, cols, [height], [width]).
    2. Press Enter to get the referenced range.

  17. What it does: Pick a value from a list based on index.

    1. Type =CHOOSE(index, value1, value2, ...).
    2. Press Enter to return the chosen value.

  18. What it does: Calculate sum, average, count, etc., while ignoring hidden rows.

    1. Type =SUBTOTAL(function_num, range).
    2. Press Enter to apply.

Analysis & Visualization (17 Techniques)

  1. What it does: Organize and view only relevant data.

    1. Select your dataset.
    2. Go to Data → Sort or Filter.
    3. Choose criteria to organize or display specific data.

  2. What it does: Summarize and analyze data quickly.

    1. Select your dataset.
    2. Go to Insert → Pivot Table.
    3. Choose where to place the Pivot Table and click OK.
    4. Drag fields to Rows, Columns, and Values areas.

  3. What it does: Create charts based on Pivot Table data.

    1. Click anywhere in the Pivot Table.
    2. Go to PivotTable Analyze → PivotChart.
    3. Choose chart type and click OK.

  4. What it does: Highlight trends or outliers in data.

    1. Select range.
    2. Go to Home → Conditional Formatting → Color Scales/Data Bars/Icons.
    3. Adjust as needed.

  5. What it does: Small charts inside cells showing trends.

    1. Select the data range.
    2. Go to Insert → Sparklines → Line/Column/Win/Loss.
    3. Set location range and click OK.

  6. What it does: Visualize data effectively.

    1. Select the data.
    2. Go to Insert → Choose chart type.
    3. Customize chart title, axis, and style.

  7. What it does: Combine two chart types for comparison.

    1. Create a chart with multiple series.
    2. Right-click → Change Chart Type → Combo.
    3. Select chart types for each series.
    4. Click OK.

  8. What it does: Find input value to achieve desired output.

    1. Go to Data → What-If Analysis → Goal Seek.
    2. Set cell, desired value, and changing cell.
    3. Click OK.

  9. What it does: Test different scenarios on your data.

    1. Go to Data → What-If Analysis → Scenario Manager.
    2. Add scenarios with different input values.
    3. Show summary to compare results.

  10. What it does: Analyze multiple input variations in a table format.

    1. Set up input cells and formula cell.
    2. Go to Data → What-If Analysis → Data Table.
    3. Specify row/column input cell and click OK.

  11. What it does: Predict future trends based on historical data.

    1. Select your time-based data.
    2. Go to Data → Forecast Sheet.
    3. Adjust settings and click Create.

  12. What it does: Quickly calculate mean, median, std deviation, etc.

    1. Enable Analysis ToolPak: File → Options → Add-Ins → Analysis ToolPak.
    2. Go to Data → Data Analysis → Descriptive Statistics.
    3. Select range and output options.
    4. Click OK.

  13. What it does: Show frequency distribution of data.

    1. Go to Data → Data Analysis → Histogram.
    2. Select input range and bin range.
    3. Choose output and click OK.

  14. What it does: Measure relationship between two variables.

    1. Go to Data → Data Analysis → Correlation.
    2. Select input range.
    3. Choose output range and click OK.

  15. What it does: Change chart appearance based on conditions.

    1. Apply conditional formatting or helper series.
    2. Format chart series to highlight conditions.

  16. What it does: Automatically update chart ranges as data grows.

    1. Define a dynamic named range using OFFSET and COUNTA.
    2. Use the name in chart series.

  17. What it does: Filter data based on complex criteria.

    1. Select your dataset.
    2. Go to Data → Advanced.
    3. Set criteria range and choose to filter in place or copy.
    4. Click OK.
